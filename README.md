## Local installation

[] Copy file .env.exampe to .env
[] change value :
        DB_CONNECTION=mysql
        DB_HOST
        DB_PORT
        DB_DATABASE
        DB_USERNAME
        DB_PASSWORD
[] run composer install
[] run npm install
[] run php artisan migrate --seed
[] run php artisan key:generate
[] run php artisan passport:install
[] run php artisan serve

use php 8.2
